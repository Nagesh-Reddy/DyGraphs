Welcome, This is a dashboard generating tool project. By using the CSV files we can create dashbaords in this project. Please have a look at the demo video.

For this project purpose we have used HighChart and Google Chart APIs as reference to generate Graphs. For now we have used 5 types of Charts.

  • Geo Chart
  • Column Chart
  • Heat Map
  • Bubble Chart
  • Scattered Chart

Following are the programming languages used to build this project :

  • Python Django
  • HTML5
  • CSS Styles
